# 🎀 Vika Agent

**Персональный AI с RAG (Retrieval-Augmented Generation)**

## Описание

Локальный AI-ассистент на базе Strategic Intelligence Engine с поддержкой RAG.

## Возможности

- 📚 **RAG** — поиск по базе знаний
- 📂 **GitHub** — работа с репозиториями
- 📝 **Анализ** — чтение кода и документов
- 💾 **Память** — сохранение контекста

## Установка

```bash
git clone https://github.com/Redrock453/vika_ok.git
cd vika_ok
pip install -r requirements.txt
```

## Запуск

```bash
python agent.py
```

или двойной клик на `run.bat`

## Команды

| Команда | Описание |
|---------|----------|
| `покажи репо` | Список репозиториев |
| `клонируй NAME` | Скачать через HTTPS |
| `клонируй ssh NAME` | Скачать через SSH |
| `анализируй репо NAME` | Анализ проекта |
| `добавь знания ФАЙЛ` | Добавить в базу |
| `найди ЗАПРОС` | Поиск в знаниях |
| `покажи знания` | Список документов |
| `очисти знания` | Удалить всё |
| `молчи` | Замолчать |
| `выход` | Выход |

## База знаний

Файлы для RAG хранятся в папке `knowledge/`.

## Безопасность

- ✅ Работает локально
- ✅ Не отправляет данные в облако
- ✅ Токены в .env (не в репозитории)
