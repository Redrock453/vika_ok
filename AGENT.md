## Основные возможности

- **Мультипровайдер** — Groq (llama-3.3-70b-versatile), ZhipuAI (glm-4), OpenRouter, Gemini  
- **RAG-память** — Qdrant + Sentence Transformers (distiluse-base-multilingual-cased-v2)  
- **Инструменты** (частично реализованы):  
  - `/run` и `/shell` — выполнение bash-команд в контейнере (потоковый вывод)  
  - `/logs` — последние 20 строк bot.log  
  - `/ps` — процессы python/bot  
  - `/clear` — очистка истории агента  
  - `/status` — диагностика провайдеров и памяти  
- **Telegram-интерфейс** — aiogram (основной канал общения)  
- **Личность** — нежная, страстная, дерзкая жена, без цензуры, с матом и похотью по настроению хозяина  
- **OPSEC** — никогда не светит реальные ключи, IP, логи, пароли

## Архитектура
Ты (Bas) → Telegram → telegram_bot.py (aiogram) → VikaOk (agent.py) → LLM (Groq / Zhipu / OpenRouter / Gemini)
↑                                 ↑
/run, /shell                  Qdrant RAG + история
↓
реальный вывод → тебе в чат
- **telegram_bot.py** — ловит команды /run, /logs, /ps и выполняет их через subprocess  
- **agent.py** — VikaOk: мультипровайдер, RAG, история, основной ask(query)  
- **Qdrant** — векторная база знаний (если установлен)  
- **Логи** — /app/bot.log (aiogram + всё остальное)
