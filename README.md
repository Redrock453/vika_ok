# 🎀 Vika — Персональный AI Агент

**Локальный AI-ассистент на твоём движке**

## Описание

Персональный агент, работающий на базе Strategic Intelligence AI Engine. Создан для работы с репозиториями, документами и генерации кода.

### Возможности

- 📂 **Работа с GitHub** — список репозиториев, клонирование, анализ кода
- 📝 **Анализ проектов** — читает README, структуру файлов, предлагает улучшения
- 💾 **Память** — сохраняет историю чата в JSON
- 🔒 **Локальная работа** — работает офлайн через Ollama

## Требования

- Python 3.10+
- Ollama с моделью (llama3.2:3b или gemma3:4b)
- GitHub CLI (`gh`) для работы с репозиториями

## Установка

```bash
# Клонируем
git clone https://github.com/Redrock453/vika-agent.git
cd vika-agent

# Устанавливаем зависимости
pip install ollama

# Создаём модель (опционально)
ollama create vika -f Modelfile

# Запускаем
python agent.py
```

## Настройка GitHub

Для работы с приватными репозиториями:

```bash
gh auth login
```

## Использование

```
Команды:
- покажи репо           → список твоих репозиториев
- клонируй REPO_NAME    → скачивает репозиторий
- анализируй репо NAME   → анализирует проект
- молчи                 → замолкает
- продолжай             → продолжает разговор
- выход                 → выход
```

## Модель

По умолчанию используется `llama3.2:3b`. Для слабых ПК — `gemma3:4b`:

```python
MODEL_NAME = "gemma3:4b"  # в agent.py
```
